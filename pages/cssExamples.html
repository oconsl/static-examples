Font properties

Font size

The font-size property is used to set the size of text in HTML. It accepts various units, such as pixels (px), ems (em), and percentages (%), among others.

For example, you can set the font size of an element to 40 pixels using the following CSS code: 

          
            span {
              font-size: 40px;
            }

This is the result of applying font-size of 40px to this tag (<span>)

Font color

The color property is used to set the color of text in HTML. It accepts various types of representation of colors such as rgb, rgba, hsl and hexadecimal.

For example, you can set the colo of the font to coral using the following CSS code: 

          
            span {
              color: coral;
            }

This is the result of applying coral color to this tag (<span>)

Font weight

The font-weight property is used to set the thickness or boldness of text in HTML. It accepts various values, such as normal, bold, bolder, and lighter, among others.

For example, you can set the font weight of an element to bold using the following CSS code: 

          
            span {
              font-weight: bold;
            }

This is the result of applying font-weight of bold to this tag (<span>)

Font family

The font-family property is used to set the font face or typeface of text in HTML. It accepts various values, such as sans-serif, serif, monospace, and cursive, among others.

For example, you can set the font family of an element to Arial using the following CSS code: 

          
            span {
              font-family: 'Trebuchet MS', 'Lucida Sans Unicode', 'Lucida Grande', 'Lucida Sans', Arial, sans-serif;
            }

This is the result of applying font-family of Arial to this tag (<span>)

            p {

Background repeat

The background-repeat property is used to control the repeating behavior of a background image. By default, a background image will repeat both horizontally and vertically to fill the entire background area of an element. However, you can use this property to change this behavior.

The possible values for this property are:

  • repeat: The background image will repeat both horizontally and vertically. This is the default value.
  • repeat-x: The background image will repeat only horizontally.
  • repeat-y: The background image will repeat only vertically.
  • no-repeat: The background image will not repeat.

For example, you can use the following CSS code to set a background image to repeat only horizontally: 

          
            p {
              background-image: url('https://i0.wp.com/imgs.hipertextual.com/wp-content/uploads/2020/10/gato-negro.jpg');
              background-size: contain;
              background-repeat: no-repeat;
              height: 100px;
            }

This is the result of applying background-repeat no-repeat to this tag (<p>)



Background position

The background-position property is used to control the position of a background image. By default, the background image is positioned at the top-left corner of an element, but you can use this property to change its position.

The property accepts two values:

  • The first value represents the horizontal position, and can be any of the following:
    • left
    • center
    • right
    • A length value, such as 10px
    • A percentage value, such as 50%
  • The second value represents the vertical position, and can be any of the following:
    • top
    • center
    • bottom
    • A length value, such as 10px
    • A percentage value, such as 50%

For example, you can use the following CSS code to set a background image and position it at the bottom-right corner of an element: 

          
            p {
              background-image: url('https://i0.wp.com/imgs.hipertextual.com/wp-content/uploads/2020/10/gato-negro.jpg');
              background-position: center center;
              background-size: contain;
              background-repeat: no-repeat;
              height: 100px;
            }

This is the result of applying background-position center center to this tag (<p>)



Background with gradient

The background property is a shorthand property that allows you to set all the background properties at once. This includes the background color, background image, and background position, as well as other properties such as background-clip, background-origin, and background-size. In addition, you can use the property to set a gradient background.

To set a gradient background, you can use the linear-gradient() function. This function accepts two or more color stops, which define the color transitions in the gradient. You can also specify the direction of the gradient using keywords such as to top or to bottom right or by specifying an angle such as 45deg.

For example, you can use the following CSS code to set a background with a linear gradient: 

          
            span {
              background: linear-gradient(90deg, #ff0000, #0000ff);
            }

This will create a gradient that transitions from coral to rebeccapurple in a direction of 90 degrees.

This is the result of applying background linear gradient to this tag (<span>)

Borders

Border

The border property is a shorthand property that allows you to set all the properties related to the border of an element at once. This includes the border width, border style, and border color.

To set the border width, you can use the border-width property. This property accepts a value or a set of values to specify the width of each side of the border. You can use the values thin, medium, and thick to set predefined widths.

To set the border style, you can use the border-style property. This property accepts a value or a set of values to specify the style of each side of the border. You can use the values solid, dashed, dotted, double, groove, ridge, inset, and outset to set predefined styles.

To set the border color, you can use the border-color property. This property accepts a value or a set of values to specify the color of each side of the border. You can use any valid CSS color value to set the border color.

For example, you can use the following CSS code to set a border with a width of 2 pixels, a solid style, and a red color: 

          
            span {
              border: 2px solid tomato;
            }

This is the result of applying a border with a width of 2 pixels, a solid style, and a tomato color to this tag (<span>)

Border radius

The border-radius property allows you to add rounded corners to an element's border. This property accepts one or more values that specify the radius of each corner.

You can use the border-radius property to create rounded corners on all four corners of an element by specifying a single value, or you can create different corner styles by specifying up to four values.

If you specify one value, it applies to all four corners. If you specify two values, the first value applies to the top-left and bottom-right corners, and the second value applies to the top-right and bottom-left corners. If you specify three values, the first value applies to the top-left corner, the second value applies to the top-right and bottom-left corners, and the third value applies to the bottom-right corner. If you specify four values, each value applies to a separate corner in the following order: top-left, top-right, bottom-right, and bottom-left.

For example, you can use the following CSS code to set a border with a radius of 10 pixels on all four corners of an element: 

          
            span {
              border: 2px solid tomato;
              border-radius: 10px;
            }

This is the result of applying a border radius of 10 pixels to all corners of this tag (<span>)

Margins

The margin property is used to create space around an element's outside edges. This property can be set to a specific value, such as 10px, or to several values to create different margins on each side of an element. You can also use the auto value to center an element horizontally within its parent element.

To set a margin to a specific value for all sides of an element, you can use the following CSS code: 

          
            span {
              margin: 100px;
            }

This will create a margin of 10px around all sides of the <span> element.

This is an example of an element with a margin of 10px.

You can also set different margins for each side of an element using the following syntax: 

          
            span {
              margin-top: 10px;
              margin-right: 60px;
              margin-bottom: 150px;
              margin-left: 400px;
            }

This will create a margin of 10px on the top of the <span> element, 60px on the right, 150px on the bottom, and 400px on the left.

This is an example of an element with different margins on each side.

Additionally, you can use the auto value to center an element horizontally within its parent element. For example, to center a <p> element horizontally, you can use the following CSS code: 

          
            p {
              width: 30%;
              margin: 0 auto;
            }

This will center the <p> element horizontally within its parent element.

This is an example of a centered element

Pseudo classes

Hover pseudo-class

The :hover pseudo-class is used to apply styles to an element when the user hovers over it with the mouse cursor. This can be useful for providing visual feedback to the user, such as changing the color of a link when it is hovered over.

To use the :hover pseudo-class, you simply add it to the end of a CSS selector. For example, the following CSS code will change the background color of all <p> elements to yellow when they are hovered over: 

          
            a:hover {
              background-color: black;
              color: white;
            }

This is a link that will change color when hovered over

Active pseudo-class

The :active pseudo-class is used to apply styles to an element while it is being activated by the user, typically when the user clicks on it with the mouse. This can be useful for providing visual feedback to the user, such as changing the color of a button when it is being pressed.

To use the :active pseudo-class, you simply add it to the end of a CSS selector. For example, the following CSS code will change the background color of all <button> elements to red while they are being clicked on: 

          
            button:active {
              background-color: black;
              color: white;
            }



Focus pseudo-class

The :focus pseudo-class is used to apply styles to an element that currently has focus, typically as a result of the user interacting with it using the keyboard or a pointing device such as a mouse or trackpad. This can be useful for providing visual feedback to the user, such as highlighting the currently focused form input or button.

To use the :focus pseudo-class, you simply add it to the end of a CSS selector. For example, the following CSS code will change the border color of all <input> elements to blue when they are in focus: 

          
            input:focus {
              background-color: black;
              color: white;
            }
